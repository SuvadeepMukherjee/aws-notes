# Fundamental Cloud Concepts

## What is Cloud Computing ? 

- Cloud Computing 
  - The on-demand delivery of IT resources over the internet with pay-as-you-go pricing 
- Benefits of Cloud Computing
  - Cost Effective
  - Scalable
  - Reliable
- Major Cloud Providers 
  - Amazon Web Services 
  - Microsoft Azure 
  - Google Cloud

## Deployment Models : Public , private and hybrid 

- Cloud Deployment Models 

  - The various ways cloud computing services are deployed and made available to users 

  - Cloud deployment models offer different levels of control and customisation over our cloud infrastructure 

  - Deployment Models 

    - Public  Cloud 

      - Cloud resources are owned and operated by a third party and services are delivered over the internet 
      - Available to multiple tenants (user)
      - Leverages a pay-as-you-go system
      - Easily accessible and no up-front infrastructure investment
      - Examples - Amazon Web Services , Microsoft Azure . Google Cloud

    - Private Cloud 

      - Dedicated to a single organisation and hosted on-premises or by a third party 
      - Offers more control , security and customization options
      - Can be implemented within an organizations data center or remotely managed 

    - Hybrid Cloud 

      - Combines elements of both public and private clouds 

    - Public vs Private Cloud 

      | Private Cloud              | Public Cloud                                 |
      | -------------------------- | -------------------------------------------- |
      | Store sensitive data       | Performing generalized tasks                 |
      | Perform regular operations | Perform operations when there is high demand |
