# AWS CloudFormation 

**Q1**:What is AWS CloudFormation ? 

**Solution**:

- Service that provides a way to manage AWS infrastructure automatically using code (Infrastructure as Code)

**Q2**:Core Components of AWS Cloudformation 

**Solution**:

- Templates
  - JSON or YAML file that defines all the AWS services
- Stacks 
  - Collection of AWS resources created and manages as a single unit using a CloudFormation template 
- Nested Stacks 
  - Nested Stacks enable us to break large templates into smaller , reusable components 
- Change Sets
  - allow us to review the proposed changes before updating existing stacks
- Stack Sets 
  - allow deployment of stacks across multiple AWS accounts from a single template
