# Amazon DynamoDB 

**Q1**:What is Amazon DynamoDB ? 

**Solution**:

- Fully managed NoSQL database service offered by AWS

**Q2**:Key Features of Amazon DynamoDB ?

**Solution**:

- Scalable
- Secure 
- High availability 
- Fully serverless

**Q3**:Common use cases of Amazon DynamoDB

**Solution**:

- Web and Mobile backends
- Serverless architecture
