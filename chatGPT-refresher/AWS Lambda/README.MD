# AWS Lambda 

**Q1**:What is AWS Lambda ? 

**Solution**:

- serverless compute service offered by AWS taht automatically runs code without requiring us to provision or manage servers 
- executes code in response to events (such as  changes in code) and automatically scales to handle any number of requests

**Q2**:Common Use cases of AWS Lambda 

**Solution**:

- Web and mobile backends 
- File uploads

**Q3**:Key features of AWS Lambda 

**Solution**:

- Scalable 
- Secure
- Cost effective 
- Event driven architecture
